2021.03学习总结

发表于2021-04-04|更新于2021-04-04|编程
|阅读量:

在 B 站开了一个装扮机器人的账号，开这个账号的目的就是学习 Python，在实践中成长无疑是最快的，所以给自己定了一些目标，是一个真·工具人账号。

月度总结

3 月关键字：咕咕咕

月初完成了基本的统计功能，以及每日的定期发布后，就没再加过新功能……

功能开发

  • 基本功能

每日统计和动态发布，基本都是自动的。毕竟 6 点的时候我还在睡觉，也就偶尔出问题了上线看下或者来吐吐槽点点关注。

  • 统计数据可视化

已经学习了常用的统计和可视化库的使用，目前只能每天统计一些，作为后续的数据支撑，毕竟没法拿到历史数据了。

等数据多了就可以上线了。但是我想全自动生成，做好能定时自动生成视频发专栏那种，这样的话需要学的东西还很多。

  • 自定义对话

简单写了一些可以自动对话的功能，没研究出私信限制的问题，没法真正用起来。算是给后续其他功能开发做个基础。比如说变身抽奖工具人。

自定义对话测试（复读机模式）

  • 编号的统计

其实要知道是谁买了 XX 号，循环读 garb/rank/fan/recent 就知道了。不过默认是会限制 IP 的，所以需要代理池。

这个已经有人开财路了我这边就不给人添堵了。毕竟不介意手动填个装扮编号的话，也就几十行代码的事。

  • 可靠性

目前使用的是 Github Actions 进行自动发布的，我还专门开了个会员。但不是特别稳定，一个月里出了 3 次事故，需要考虑如何改进。

  • 站外数据

深深感受到站内的限制太多，而且指不定一绝醒来这个号就没了。所以也考虑搭个小站这样也自由一些。一想到 VPS 要钱，域名要钱，深深感受到贫穷。。

学习计划

√ Python基础语法

√ B 站 App 和网页版的抓*方法

√ 使用开源库如 Bilibili_API 进行二次开发

√ 使用 requests 库 + 抓包原生开发

× 代理池

× 数据存储和分析

× 数据可视化

× 统计视频自动生成

× 研究下 B 站的访问限制策略

√ 使用 github actions 进行持续发布

× 灾备服务

× 重构并开源代码

√ 博客网站搭建

× Vue

× QQBot

× 微信/QQ/企业微信/钉钉 推送服务

√ 库存提醒工具 (Windows / Linux / MacOS)

× 库存提醒工具安卓 App 开发

× 库存提醒工具 IOS App 开发

× 个人所有装扮编号统计工具开发

× 装扮圈词云

总结

穷则独善其身，达则兼济天下。

这里只是一个没有存在感的工具人，一个期望保持理智的观察者。
